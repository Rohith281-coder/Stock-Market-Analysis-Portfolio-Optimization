# Stock Market Analysis & Portfolio Optimization

## Overview

This project analyzes historical stock market data using Python and SQL to evaluate stock performance, measure risk, and optimize portfolio allocation. The objective is to help investors make data-driven investment decisions through return analysis, volatility measurement, and diversification strategies.

## Objectives

* Analyze historical stock price performance.
* Calculate daily and cumulative returns.
* Measure stock volatility and investment risk.
* Evaluate correlations between stocks.
* Demonstrate the benefits of portfolio diversification.
* Generate actionable insights for portfolio optimization.

## Key Findings

* Diversified portfolios reduced overall investment risk by approximately 30%.
* High-return stocks generally exhibited higher volatility.
* Correlation analysis identified stocks suitable for diversification.
* Long-term investment strategies produced more stable returns than concentrating investments in a single stock.
